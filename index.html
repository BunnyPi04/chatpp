About Chat++

Chat++ is an all-in-one Chatwork Enhancement Toolkit. It is a free Chrome Extension which provides more funny emoticons as well as many other features for Chatwork. It is made by Framgia's Members, and for Framgia's Members.

Chat++ is a big innovation of the well known extension YACEP!

Download Chat++

You can download and install Chat++ via the Google Chrome Web Store!

Create Your Own Emoticons Data

You can create your own Emoticons data file and then share it with your friends. Of course if you and your friends have the same data, you will see the same emoticons on Chatwork! You can easily create a Emoticons Data file for Chat++ by following the below rules:

  • The data file (as well as the change log file) must be written in json format.
  • The data file must have the following data:
    • data_name: Your data name (It should not be updated when you update your data file)
    • data_version: Your data version (It should be updated when you update your data file)
    • data_changelog (optional): The link to where the change log file is hosted
    • emoticons: Emoticons data. It is an array with each element is an emoticon's data.
    • You can refer the default emoticons data as well as the change log file of YACEP for more details.
  • You need to host your data file on the internet so Chat++ can access it! (https is required! So we suggest Dropbox!)
  • Finally, share the link to your data file with your friends and enjoy your own funny Emoticons!
